# RCCG_71st_Annual_Convention_Sermons
This repository contains sermons from the 71st Annual convention of the Redeemed Christian Church of God, in form of Jupyter notebooks and PDFs

To include essential points that you feel are ommitted, simply download the .pynb file, make the necessary additions and send a pull request. Additionally, an easier alternative would be that you send an email to _sermons.beyondexpectations@gmail.com_ with the additions and it will be considered upon vetting.
